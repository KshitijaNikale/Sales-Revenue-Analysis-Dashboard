# 📊 Sales & Revenue Analysis Dashboard

## Overview
An interactive Power BI dashboard built using the Superstore dataset to analyze sales performance.

## Tools Used
- Power BI
- DAX
- Excel
  
## Features
- Total Sales
- Total Profit
- Total Orders
- Total Quantity
- Monthly Sales Trend
- Sales by Category
- Profit by Region
- Sales by Segment
- Interactive Filters
